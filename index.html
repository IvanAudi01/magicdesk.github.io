Misión de
Magic Desk Support

Nosotros tenemos la misión de poder apoyar a usuarios con problemas puntuales,
problemas que pueden ser resueltos de la manera más pronta posible
para que el cliente final no tenga retrasos en su producción

  • Inicio

Idea de creación
del proyecto

Todo comenzó ya que usuarios utilizaban una mesa de ayuda para atención a usuarios de esta manera pensamos en realizar nuestra propia mesa de ayuda y ser propiertarios de ella para poder darle el diseño y funcionamiento a nuestro gusto

Funcionamiento del
proyecto

Nosotros pensamos en crearlo de una manera sencilla, que pudiera ser intuitiva para los usuarios finales esperando y pensando que no todos tienen una habilidad perfecta de utilizar un equipo electrónico.


Desarrollo del
proyecto

El proyecto de desarrollé pensando en las necesidades de los usuarios. Fácil entendimiento, uso simple, interfaz amigable. Realizamos encuentas a los ususarios para poder entenderlos y hacerlo mejor posible para ellos.

Herramientas de
trabajo

Nosotros utilizamos varias herramientas de trabajo entre ellos: Python, HTML, CSS, PHP, JS, entre otros.




Funcionamiento del
sistema

El sistema se muestra con una interfaz súper amigable, la cuál solo presenta algunos accesos para que los usuarios puedan levantar sus tickets. Especificando la problemática que se les presenta y poder adjuntar imágnes al respecto. Se responde una vez haya quedado resuelto por parte del soporte y se puede cerrar el ticket en caso de que se haya resuelto.

Vista final del
proyecto

Podemos ver de manera superficial la interfaz del usuario y sus funcionalidades. Lo que se presenta a continuación es la vista del perfil de soporte que nos muestra la cantidad de tickets y la información que presenta cada uno de ellos.
